# Fake-LoginPanel-Flooder
 A program to flood a fake login panel with random passwords. I made this program to spam a Steam account stealer site called casenorm.fun. It was like hellcase.com, but they stole my friend's steam account. Fortunately he got back his account. Steam admins were very nice. And it worked. I spammed them over 1000 tries and the site is still not online.
